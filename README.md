# simple-web-app
Mockup web sederhana menggunakan [materialize css](https://materializecss.com)

### Cara menggunakan project
- Clone kode project:
```bash
$ git clone https://github.com/sutrisna/simple-web-app
$ cd simple-web-app
$ Buka index.html dengan klik 2x pada file tersebut
```
